# Church Bingo 🎱

A Bingo application built for my local church community.
Designed to be used with a physical ball machine — the operator
enters the number called and the program tracks it automatically.

## Features
- Automatic number tracking
- Visual bingo board
- Extra control buttons
- Built with C# and WPF (.NET)

## Tech Stack
- Language: C#
- Framework: WPF
- IDE: Visual Studio
