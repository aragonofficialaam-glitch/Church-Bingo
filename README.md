# Church Bingo 🎱

A Bingo application built for my local church community, designed to work alongside a physical ball machine. The operator enters the number called and the program tracks and displays it automatically.

## Features
- Automatic number tracking
- Visual bingo board with highlighted numbers
- Call history log
- High-visibility colors for projector display
- Extra control buttons for easy operation

## Tech Stack
- **Language:** C#
- **Framework:** WPF (.NET)
- **IDE:** Visual Studio 2026

## How it works
The operator moves the physical bingo balls and enters the number into the program. The app highlights the number on the board and keeps a history of all called numbers during the session.
